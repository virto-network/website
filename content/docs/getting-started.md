# Getting Started

This guide will help you set up VirtoSDK and make your first API call.

## Installation

```bash
npm install @virto/sdk
```

## Basic Usage

```typescript
import { createClient } from '@virto/sdk'

const virto = createClient({
  network: 'kreivo',
})
```
